deStyle - A Blog Magazine Theme by ThemeShift

Thanks for Using deStyle!

Welcome and thanks for using my blog magazine WordPress theme deStyle.

Installation

  1. Download deStyle zip archive.
  2. Go to WP-Admin > Appearance > Install Themes > Upload and choose the new theme zip folder.

To find out more about installing WordPress themes please also see the WordPress documentation.

Update

  1. Download the new version of the theme.
  2. Deactivate the current version of the theme and remove the files from your server. (*)
  3. Go to WP-Admin > Appearance > Install Themes > Upload and choose the new theme zip folder.

(*) Backup your old theme files! Any changes you have made to the theme templates will be lost. Please consider using Child Themes.

Post / Page Images

The theme supports WordPress post thumbnails. To add the main post image please click on Set featured image in the Featured Image box on the right, upload your image and click Use as featured image. deStyle will automatically create all the necessary image sizes.

Upload Custom Logo

On the theme options page you can insert the URL or directly upload your custom logo.

  1. Click on Upload Image
  2. Choose the image file from your computer
  3. Click on Media Library
  4. Click on Show
  5. Click on Insert into Post *

* Please make sure that the size Full Size is selected.

Read More Buttons

To cut off posts and create a read more button please use the WordPress more tag. You'll find this feature in the tool bar of the WordPress post editor. Please find some more information.

Theme Options

In order to customize deStyle to your needs please visit WP-Admin > deStyle > Theme Options in the WP-Admin menu and set your options.

General

  1. Logo

    Please enter an URL or upload your custom logo from your local computer.

  2. RSS URL

    Please enter your custom RSS URL (e.g. Feedburner).

  3. Twitter

    Please enter your Twitter user (e.g. themeshift).

  4. Custom CSS

    Easily add some custom css to the head of your theme.

  5. Tracking Code

    Insert your tracking code here (e.g. Google Analytics). This code will be added to the footer of the theme.

Layout

  1. Color Scheme

    Please select your preferred color scheme.

  2. Color Switcher

    Tick the box to display a jQuery color switcher.

  3. Background

    Please select your preferred background. If you want to use the WordPress custom background feature on WP-Admin > Appearance > Background, please choose None.

Menu

  1. Home

    Exclude the home link from the menu. The logo will also be linked to the home page.

  2. Categories

    Please enter the category IDs you want to exlcude from the menu (e.g. 2,5,9).

  3. Pages

    Please enter the page IDs you want to exlcude from the menu (e.g. 3,65,71).

Sidebar Ads

  1. Hide Ads

    Check the box to hide the follwing ads from the sidebar without loosing any inserted information.

  2. Ad - Image

    Please enter image url of a square button (125x125px).

  3. Ad - Link

    Please enter link url of this ad.

  4. Ad - Nofollow

    Check the box to remove nofollow of this ad.
